## 4. 关于试用版

```
0. 直接下载即为试用版
1. 试用版会随机混淆30%左右的代码
2. 剩余70%部分只是简单的添加前后缀字母
```

例如：
```
- (void)hello;//方法

随机被混淆:
- (void)xxx;//混淆结果未知

随机不被混淆:
- (void)aHelloB;//大概是这样,只会有前后随机字母的差别
```
